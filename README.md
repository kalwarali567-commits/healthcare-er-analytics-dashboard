# Healthcare Emergency Room Analytics Dashboard

## Project Overview
This project analyzes hospital emergency room data to identify operational inefficiencies, patient satisfaction trends, admission patterns, and financial risks. The analysis was performed using SQL and Power BI to support healthcare management decision-making.

## Business Problem
Hospitals often face:
- Long patient waiting times
- Delayed insurance payments
- High operational workload
- Low patient satisfaction
- Resource allocation challenges

This project aims to analyze these problems using healthcare data analytics techniques.
## Tools Used
- SQL
- Power BI
- CSV Dataset
- Data Visualization
- Healthcare Analytics

---

## Dataset Information
The dataset contains emergency room patient visit records including:
- Patient demographics
- Department details
- Wait times
- Diagnosis information
- Billing and insurance data
- Patient satisfaction scores
- Admission records

---

## Key Analysis Performed

### 1. Revenue Analysis
- Compared paid vs pending payments
- Identified cash flow risks

### 2. Wait Time Analysis
- Measured average patient waiting time by shift
- Identified operational bottlenecks

### 3. Patient Satisfaction Analysis
- Compared satisfaction scores across departments
- Highlighted areas needing service improvement

### 4. Admission Analysis
- Identified diagnoses causing high hospital admissions
- Supported healthcare resource planning

### 5. Insurance Payment Analysis
- Analyzed pending payments by insurance provider
- Evaluated revenue cycle inefficiencies

---

## Key Insights
- Night shifts showed higher patient waiting times
- Emergency departments had lower satisfaction scores
- Pending insurance payments affected hospital cash flow
- Critical diagnoses increased operational workload

---

## Recommendations
- Improve emergency room staffing during night shifts
- Optimize billing and insurance claim workflows
- Reduce patient waiting times
- Improve patient experience monitoring
- Strengthen operational resource allocation
